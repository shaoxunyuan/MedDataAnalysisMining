线性回归与模型诊断

本练习使用 MASS::Melanoma：以连续结局 thickness 为例，练习 lm() 拟合、系数解释、均值区间与个体预测区间、嵌套模型比较，以及基础诊断（残差形态、正态性、同方差、杠杆与影响点）。
学习目标
会拟合简单与多重线性回归；能区分条件均值预测与个体预测区间；能用嵌套模型 ANOVA 做增量解释；能读懂诊断图并列出不夸大因果的结论要点

0. 载入 Melanoma

0.1 载入数据

options(repos = c(CRAN = "https://mirrors.tuna.tsinghua.edu.cn/CRAN/"))
if (!requireNamespace("MASS", quietly = TRUE)) install.packages("MASS")
library(MASS)
data(Melanoma)
mel = as.data.frame(Melanoma)

1. 简单线性回归

1.1 拟合与系数解释

模型：用 age 解释 thickness，形式为 thickness ~ age。
因果表述：线性回归刻画的是关联（association），不是随机对照试验下的因果效应。除非你有明确的因果设计与识别策略，否则结论宜用“在控制了……之后，与……相关联”这类措辞。
Step 1 / 拟合与查看摘要
fit1 = lm(thickness ~ age, data = mel)
summary(fit1)
Step 1b / 全局检验：模型是否有解释力（F 检验）
anova(fit1)
怎么读：summary(fit1) 底部通常会给出 “F-statistic” 与对应 p 值，含义是在检验“除了截距以外，所有斜率系数同时为 0”的联合原假设；只有一个斜率时与 anova(fit1) 中 age 行一致。
Step 2 / 散点图 + 回归线
plot(mel$age, mel$thickness,
     xlab = "age", ylab = "thickness",
     main = "thickness ~ age",
     pch = 19, col = rgb(13/255, 148/255, 136/255, 0.35))
abline(fit1, col = "#b91c1c", lwd = 2)
Step 2b /（可选）标准化系数：比较不同尺度变量的“相对强度”
fit1_z = lm(scale(thickness) ~ scale(age), data = mel)
coef(fit1_z)
# scale(thickness) 与 scale(age) 的均值约为 0、标准差约为 1；斜率可粗略理解为“标准化单位的关联强度”

1.2 置信区间与预测

Step 3 / 系数置信区间
confint(fit1)
Step 4 / 给定年龄的预测（点预测）
newd = data.frame(age = c(30, 50, 70))
predict(fit1, newdata = newd)
Step 4b / 区间预测：条件均值 CI vs 个体响应 PI
newd = data.frame(age = c(30, 50, 70))

# 条件均值 E(Y|X) 的置信区间（更窄）：回答“回归线在这一点附近的不确定性”
predict(fit1, newdata = newd, interval = "confidence", level = 0.95)

# 个体 Y 的预测区间（更宽）：回答“一个新个体的 thickness 可能落在哪”
predict(fit1, newdata = newd, interval = "prediction", level = 0.95)

# 残差标准差（RMSE）sigma_hat：summary(fit1)$sigma
summary(fit1)$sigma
写作模板：报告预测时最好写明你在预测均值还是个体；同一模型下，个体预测区间通常明显宽于均值置信区间。

2. 多重线性回归与诊断

2.1 加入协变量与分类变量

示例：加入 ulcer 与 sex（二分类变量）以及 year。其中 sex 在原数据里编码为 0/1，按分类变量进入模型更符合解释习惯。
Step 5 / 多重线性回归
fit2 = lm(thickness ~ age + factor(sex) + ulcer + year, data = mel)
summary(fit2)
confint(fit2)
Step 5b / 对照 levels：确认因子基准水平（决定截距与哑变量的符号）
levels(factor(mel$sex))
# factor(sex) 会以字典序/levels 的第一个水平作为基准（reference）
Step 5c / 数值预测变量的线性关联强度（粗看多重共线性）
cor(mel[, c("age", "thickness", "year", "ulcer")], use = "complete.obs")
解读提示：两预测变量之间相关性很高时，系数会更不稳定（标准误变大），这时更适合讨论“组合效应/模型整体”，或对变量做领域驱动的取舍。
Step 5d / 嵌套模型比较：fit2 相对 fit1 是否显著改进
anova(fit1, fit2)
结果：这是一般线性模型的嵌套比较（fit1 变量 ⊆ fit2），可用最后一行的 F 与 Pr(>F) 判断新增变量整体是否带来显著增量解释。

2.2 诊断图与影响点

经典假设（便于对照看图）：近似线性关系；误差独立；给定预测变量条件下误差近似正态；方差齐性（同方差）。现实数据常常是“近似满足”，诊断目的是发现明显违背而不是追求完美。
Step 6 / 四张经典诊断图
par(mfrow = c(2, 2))
plot(fit2, which = 1:4)
par(mfrow = c(1, 1))
读图要点（各看一张就够入门）：Residuals vs Fitted 看是否存在系统性弯曲（提示非线性或缺失交互）；Normal Q-Q 看尾部偏离（厚尾/极端值）；Scale-Location 看残差散布是否随拟合值扩大（异方差线索）；Residuals vs Leverage 结合 Cook’s distance 轮廓线识别高杠杆且高残差的点。
Step 7 / Cook 距离：查看影响点（数值最大的若干条）
ck = cooks.distance(fit2)
head(sort(ck, decreasing = TRUE), 8)
Step 7b / 把影响点定位回数据行（便于核对）
top = order(ck, decreasing = TRUE)[1:5]
cbind(row_index = top, cook = ck[top], mel[top, c("time", "status", "sex", "age", "year", "thickness", "ulcer")])
注意：影响点不等于“必须删除”。通常需要回到原始记录核对：是否录入问题、是否极端病例、是否提示模型形式需要调整（变换/加项/稳健方法）。

3. 实验内容（提交要求）

提交要求：按下列条目给出相应输出与简要说明。
A. 简单线性回归
  • 模型输出：提交 summary(fit1) 的关键行（系数表与 R²）。
  • 全局检验：提交 anova(fit1) 中与模型相关的 F/p（或 summary 底部 F 统计量），并用一句话说明它在检验什么。
  • 解释：用一句话解释 age 的回归系数代表什么。
  • 区间预测：对 age=50 同时给出 interval="confidence" 与 interval="prediction" 的 95% 区间，并用一句话解释二者宽窄差异的含义。
  • 图：提交图 1（或同类图）截图。
B. 多重回归与诊断
  • 模型输出：提交 summary(fit2) 的关键行，并说明与 fit1 相比你观察到的 1 个差异（系数变化或解释差异）。
  • 嵌套比较：提交 anova(fit1, fit2) 的最后一行，并用一句话给出结论（新增变量整体是否显著）。
  • 因子基准：说明你模型里 factor(sex) 的基准水平是哪个编码值，以及另一个水平的系数代表什么。
  • 诊断：提交图 2（或同类诊断图）截图，并指出你认为最需要关注的一张（1 句话）。
  • 影响点：提交 Cook 距离最大的前 3 个值，并说明你会如何处理/核对（1–2 句话）。
